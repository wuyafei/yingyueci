常见问题

如何导入 EPUB？

进入“书架”，点击左上角加号，选择本地 EPUB 文件导入。

如何导入英文文章？

进入“文章”，点击左上角加号，输入可访问的英文文章 URL，应用会尝试抓取正文并导入阅读。

为什么在线翻译或 AI 翻译不可用？

请进入“翻译与 AI 设置”，配置对应供应商的 API Key 和模型名称。未配置 API Key 时，相关在线能力不会调用。

我的 API Key 存在哪里？

API Key 保存在设备本地的 iOS Keychain 中。开发者不会收集或保存你的 API Key。

生词本可以导出吗？

可以。进入“生词本”，使用导出功能生成生词文件。

问题反馈建议

  • 无法导入 EPUB：请说明文件来源、文件大小，以及是否能被其他阅读器打开。
  • 文章抓取失败：请提供文章 URL，并确认网页可在浏览器中访问。
  • 翻译失败：请注明使用的翻译供应商、模型名称和错误提示。
  • 阅读或查词异常：请提供具体书籍章节、单词和操作步骤。
