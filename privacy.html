隐私声明

本声明说明英阅词如何处理用户数据。最后更新：2026 年 5 月 23 日。

概述

英阅词尊重用户隐私。应用不提供账号系统，不主动收集用户身份信息，不包含广告追踪 SDK，也不将用户阅读数据上传到开发者服务器。

本地数据

为了提供阅读、查词、生词本和复习功能，英阅词会在设备本地保存以下数据：

  • 用户导入的 EPUB 文件、英文文章内容和阅读进度。
  • 用户保存的生词、释义、原文例句、复习状态和相关设置。
  • 翻译供应商配置、模型名称和应用偏好设置。

这些数据默认保存在用户设备本地。删除 App 可能会同时删除这些本地数据，具体取决于 iOS 的系统行为和用户备份设置。

API Key 与在线能力

如果用户启用在线翻译或 AI 功能，需要自行填写对应供应商的 API Key。API Key 保存在设备本地的 iOS Keychain 中，开发者不会收集或保存用户的 API Key。

当用户主动使用在线翻译、AI 翻译或 AI 增强复习时，应用会把必要的文本内容发送给用户选择的服务供应商，用于完成本次请求。这些供应商可能包括腾讯云翻译、智谱 GLM、DeepSeek、MiniMax、Kimi 或用户配置的相关模型服务。

第三方服务对数据的处理受其自身服务条款和隐私政策约束。用户应在配置 API Key 前了解并同意对应供应商的政策。

文章导入

当用户在“文章”中输入 URL 并主动导入时，应用会访问该 URL，尝试抓取网页正文用于本地阅读。应用不会自动扫描用户浏览记录，也不会在用户未操作时抓取网页内容。

不会收集的数据

  • 不会收集用户姓名、手机号、邮箱等身份信息。
  • 不会收集精确位置信息。
  • 不会收集通讯录、照片、麦克风、摄像头等敏感权限数据。
  • 不会出于广告目的跟踪用户。

数据删除

用户可以在 App 内删除书籍、文章或生词数据。用户也可以通过 iOS 系统删除 App，从设备中移除英阅词保存的本地数据。

如果用户已在第三方供应商处产生在线翻译或 AI 请求记录，请按对应供应商提供的方式管理或删除相关数据。

儿童隐私

英阅词不面向儿童主动收集个人信息。如果监护人认为儿童向第三方供应商提交了个人信息，请联系对应供应商处理。

政策更新

当应用功能或数据处理方式发生变化时，本隐私声明可能会更新。更新后的声明会发布在本页面。
